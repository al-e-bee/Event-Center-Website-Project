# The Woodward Exchange - Event Center Project

A sophisticated, responsive multi-page website built as a project for Coding Temple. **The Woodward Exchange** is a high-end event venue located in the heart of Detroit, focusing on elegance, strength, and professional service.

## 🚀 Live Features

- **Multi-Page Architecture:** Includes Home, About, Events, and a dedicated High-Res Gallery.
- **Hybrid Navigation:** Combines external page links with internal anchor links (`#testimonials`, `#contact`) for a seamless user experience.
- **Responsive Design:** Custom media queries for Mobile (< 600px), Tablet (600px - 1024px), and Ultra-Wide Desktop (1920px+).
- **Interactive Elements:** JavaScript-powered mobile "hamburger" menu and a "Back to Top" scrolling button.

## 🛠️ Technical Implementation (Rubric Compliance)

- **Semantic HTML5:** Used `<header>`, `<main>`, `<section>`, `<figure>`, and `<footer>` for SEO and accessibility.
- **CSS Layouts:** Utilized **Flexbox** for the navigation and footer, and **CSS Grid** for the Team section and Image Galleries.
- **Image Formats:** \* `.jpg`: High-resolution hero and gallery imagery.
  - `.png`: Transparent brand assets (Lion Logo).
- **Data Organization:** Implemented a styled HTML `<table>` for the 2026 Event Schedule and an `<ol>` for the Booking Process steps.
- **Forms:** Fully styled Contact/Booking form with custom input focus states and backdrop-filter blurring.

## 🎨 Design Choices

- **Typography:** A pairing of _Playfair Display_ (Serif) for a classic, luxurious feel and _Inter/Roboto_ (Sans-serif) for modern readability.
- **Color Palette:** \* Primary: `#222725` (Deep Charcoal)
  - Accent: `#e59500` (Metallic Gold)
  - Background: `#f7f7f2` (Soft Off-White)
- **Visual Identity:** The "Lion with Crown" logo represents the "Strength and Elegance" theme mentioned in the project brief.
